# 🐾 Хатинка лапок

### 📖 Про проєкт

«Хатинка лапок» — це навчальний командний front-end проєкт, створений у рамках
курсу Full-Stack Developer.

> Проєкт являє собою вебсайт, присвячений допомозі тваринам знайти затишну
> домівку, інформуванню користувачів та >демонстрації сучасних підходів до
> верстки й JavaScript-розробки. Сайт містить адаптивний
> інтерфейс, >інтерактивні елементи та динамічний контент.

### 🛠️ Технології та інструменти

🌐 HTML — семантична розмітка сторінки. 🎨 CSS — стилізація та адаптивний
дизайн. ⚡ JavaScript — інтерактивність та логіка роботи сайту. 🚀 Vite —
швидкий збірник та середовище розробки. 🔧 Git — контроль версій та командна
робота. ⚙️ Backend — REST API та пагінація

### 📚 Бібліотеки:

📸 Swiper.js — слайдер для галерей та каруселей. 🌐 Axios — робота з
HTTP-запитами. 🔔 sweetalert2 — стильні повідомлення та нотифікації. 📂
Accordion — інтерактивні блоки для FAQ та секцій.

### 🚀 Запуск проєкту локально

Клонувати репозиторій:

```sh
git clone
```

Встановити залежності:

```sh
npm install
```

Запустити проєкт у режимі розробки:

```sh
npm run dev
```

Відкрити проєкт у браузері:

```sh
http://localhost:5173
```
